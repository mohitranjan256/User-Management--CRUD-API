
# USER-MANAGEMENT 

Backened Server : 
 1. Upload a file
 2. Get list of all the files
 3. Download a file
 4. Update a file
 5. Delete a file
 


## Tech Stack


**Server:** Node, Express

**Database:** MongoDB


## Run Locally

Clone the project

```bash
  git clone https://github.com/monster256/webteam-task-3.git
```

Go to the project directory

```bash
  cd webteam-task-3
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon index.js
```
